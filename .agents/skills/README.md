# SafeCampus Skills

This folder contains local skills for coding agents working in the SafeCampus monorepo.

How to use:
- Pick the skill that matches the current task.
- Follow the checklist in the skill `SKILL.md`.
- Use project commands from this repo, not generic templates.

Installed skills:
- `turborepo`
- `next-best-practices`
- `tailwind-v4-shadcn`
- `vercel-react-best-practices`
- `supabase-postgres-best-practices`
- `fastapi-clean-architecture`
- `testing-monorepo`
- `monorepo-ci-standards`
